# transcribe-service

## Описание

Этот проект представляет собой сервис для транскрибирования аудиофайлов в текст. Основная цель — предоставить простой API для загрузки аудио и получения текстовой расшифровки.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/transcribe-service.git
   cd transcribe-service
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

Запустите основной файл:
```bash
python main.py
```

## Структура проекта
- `main.py` — основной файл запуска сервиса
- `config.py` — конфигурация проекта
- `requirements.txt` — зависимости Python

## Требования
- Python 3.8+
- Все зависимости из `requirements.txt`
